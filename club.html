About

Whether you're planning on becoming a physicist,
or you just enjoy doing math, we have lectures
made to prepare you for whatever you need. We
will also do lectures to help you get ahead in
the Sidwell math curriculum too.

This Week's Lectures

On Thursday, we'll start Single Variable calculus
since it's needed for the rest of the lectures
this year. We'll start with limits, and then move
into differentiation and antiderivatives. This will
set us up to go into basic Linear Algebra next, and then
begin modeling more complex systems using equations.

On Monday to prep for the USACO, I'm doing lectures
on Big O, Big Theta, and Big Omega time complexity
and if there's time we'll start to cover some
advanced but useful data structures & optimization
techniques.
